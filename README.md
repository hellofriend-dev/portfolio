### Технические детали

Каждое демо — самодостаточный `index.html`:

- ноль зависимостей, единственный внешний ресурс — шрифт Inter с Google Fonts;
- анимации появления на `IntersectionObserver`, без анимационных библиотек;
- адаптив от 320 px, поддержка `prefers-reduced-motion`;
- страницы остаются читаемыми при отключённом JavaScript.

Единая дизайн-система: тёмный минимализм, фон `#0A0A0A`, акцент `#6C63FF`, волосяные границы вместо теней.
